Este programa necesita acceso a la raíz para funcionar (Tu dispositivo debe estar rooteado)

Descripción:
-----------------
Verde - Esta aplicación tiene gran probabilidad de ser registrado;
Amarillo - Existe un Parche Específico para esta aplicación en la carpeta /sdcard/LuckyPatcher/;
Azul - Esta aplicación tiene Anuncios (Google Ads);
Violeta - Esta aplicación está en la Lista de Arranque;
Rojo - Esta aplicación no se puede registrar. A excepción de que el programa se componga de dos partes (Aplicación principal y Aplicación de registro -APK Pro Key-), en cuyo caso, debes aplicar el parche a ambas aplicaciones, no sólo a la principal.;
Naranja - Aplicación del Sistema. Ten mucho cuidado al realizar cualquier operación con ella, puede resultar perjudicial para la aplicación o el sistema operativo.;
Lista de Arranque - Contiene la lista de programas que serán parchados cuándo el dispositivo inicie. Se utiliza sólo para aplicaciones que necesitan correción de bibliotecas ".so", pero que no existe un parche específico para las mismas; exceptuando versiones antiguas de Titanium backup, en las que no puede utilizarse;

Estrella - Indica que el dalvik-cache de la aplicación ha sido corregido (ODEX con cambios creados). Esto significa que la aplicación se mantendrá en su estado actual hasta que sea actualizada o eliminada. El ODEX puede ser removido con la opción "Remover ODEX con cambios". Ésto restaurará la aplicación a su estado original como antes de ser parchada. Cuando actualizas una aplicación puede surgir el siguiente problema: Google Play notificará que no queda espacio suficiente, para solucionarlo ejecuta Lucky Patcher, luego cierra y vuelve a intentar instalar la aplicación, Si ese mismo error persiste o aparece al momento de desinstalar la aplicación selecciona en las configuraciones de Luckypatcher "Solución de problemas->Eliminar correciones y copias de seguridad (Para actualizar la App)" - Esto eliminará todas las correcciones del dalvik-cache que habían sido realizadas;

Context Menu:
-----------------
Remover la verificación de licencias (Modo automático) - Intenta eliminar la verificación de licencia por parte de Google Play. Utiliza una pequeña parte de las plantillas para asegurar la máxima estabilidad de la aplicación
Remover la verificación de licencias (Modo automático Inverso) - Existen alguna diferencias respecto al "Modo Automático". Puede ayudar en caso de que ayude el "Modo Automático".
Remover la verificación de licencias (Modo Automático Extremo) - Intenta elimar la verificación de la licencia utilizando todas las plantillas. La aplicación puede volverse inestable. La mayoría son errores de conexión.
Remover la verificación de licencias (Modo Manual) - . Remueve la licencia por fuerza bruta.
Remover la verificación de licencias (Seleccionando Patrones) - Puedes seleccionar el grupo de plantillas a modificar.
Remove Anuncios (Google Ads!) - Intenta remover los anuncios (Google Ads). Algunas veces al remover los anuncios, la aplicación puede mostrar una leyenda diciendo que el anuncio no ha sido encontrado (Not found!)
Remover ODEX con Cambios - Remueve el archivo ODEX manteniendo los cambios. Restaurando así el estado original de las aplicaciones.
Modificar Permisos - Remueve los Permisos de acceso de las Aplicaciones. Ten mucho cuidado al modificar estos permisos. Puede producir ciclos de reinicios contínuos u otras desagradables consecuencias. Asegurate de hacer un resplando del sistema antes de eliminar permisos.
Crear un APK modificado - Crea un Paquete de Instalación APK en base a los parches aplicados. Debes saber que el archivo creado no contiene los parches aplicados . Dado que parchar una aplicación es muy fácil para probar tu mismo los cambios.
Parchar al reiniciar - Utilizalo sólo para aplicaciones que necesitan correción de bibliotecas ".so", en las que no existe un parche específico para las mismas; exceptuando versiones antiguas de Titanium backup, en las que no puede utilizarse. Trabaja muy inestable. Es recomendable no aplicar más de un Parche Específico por aplicación. Si fuera necesario, se escribirá uno.
Parchar manualmente - Parcha la aplicación para depurarla.
Copia de seguridad! - Crea una copia de respaldo (apk) para poder resturar la aplicación. El archivo Apk se guarda en /sdcard/LuckyPatcher/Backup/.
Restaurar! - Restaura la aplicación desde la carpeta /sdcard/LuckyPatcher/Backup/.


Copiar cambios al dalvik-cache! - En el modo por defecto, todos los cambios permanecen en el odex-file. Algunos odex-file no trabajan para la aplicación (Cuando una aplicación es movida a la SD). Entonces copiar el dalvik-cache puede ayudar. Pero puede restaurar la aplicación sólo reinstándola.
Respaldar en archivo apk para reinstalar! - Copia el archivo apk a la microSD (sdcard) dentro de la carpeta Data del Lucky Patcher. Puedes reinstalar (Restaurar) esta aplicación desde el archivo apk.
Borar dalvik-cache! - Después de crear el odex-file borra el dalvik-cache de esta aplicación para ahorrar espacio. Si la aplicación no trabaja con odex, tendrás que reinstalarla.

Instrucciones:
--------------
Para remover la verificación de licencias:
- Se usa en archivos apk originales firmados por el desarrollador;
- Manten presionado sobre la aplicación y selecciona "Remover la verificación de licencias";
- Selecciona "Modo automático" y prueba la aplicación con acceso a internet;
- Si no funcionó, selecciona "Modo Automático Inverso" o "Modo Automático Extremo";
- Si tienes suerte, el programa será registrado;
- Para regresar la aplicación a su estado original, necesitas presionar "Remover ODEX con cambios" o "Restaurar".

Para remover los anuncios (Google Ads):
- Manten presionado sobre la aplicación y selecciona "Remover anuncios (Google Ads)";
- Ejecuta y prueba la aplicación con Internet;
- Si tienes suerte, los anuncios ya no aparecerán;
- Para regresar la aplicación a su estado original, necesitas presionar "Remover ODEX con cambios" o "Restaurar".

Para aplicar un parche específico:
- Mantén presionado sobre la aplicación y selecciona "Parche específico!";
- Ejecuta y prueba la aplicación con internet;
- Si tienes suerte, el parche funcionará;
- Para regresar la aplicación a si estado original, debes presionar "Remover ODEX con cambios" o "Restaurar".

Para cambiar permisos:
- Presiona en el Software y selecciona "Cambiar permisos";
- Presiona permisos para desactivar (rojo);
- Presiona permisos para activar (verde);
- Debes presionar Aplicar (Reiniciar).

Para crear un APK modificado:
- Presiona el Software y selecciona "Crear APK modificado";
- Selecciona el parche para modificar el APK;
- El archivo de instalación modificado se guardará en la carpeta /sdcard/LuckPatcher/Modified/ ;
- Debes saber que el archivo creado no contiene los parches aplicados . Dado que parchar una aplicación es muy fácil para probar tu mismo los cambios.

PS: El parche provoca que la aplicación se cierre, al momento de aplicarse. Esto es necesario para que lo cambios sean aplicados. Pero aún así en ocasiones es necesario que reinicies el dispositivo o que finalices la aplicacion desde el administrador de aplicaciones.
